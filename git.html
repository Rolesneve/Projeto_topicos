Controle de versão:O Git se refere a uma ferramenta de controle de versão, usada principalmente para auxiliar na criação e desenvolvimento de projetos, que pode ter alta ou baixa complexidade. Utilizando uma ferramenta como essa se torna possível acompanhar as diversas etapas de criação de um programa já que o mesmo usa de ferramentas que fazem isso ser possivel. Exemplificando: Digamos que você tem interesse de voltar em alguma versão antiga do seu projeto em vez de desfazer o que já está feito é muito mais simples ver uma versão antiga que já está salva. Essa ferramenta também torna possível que diversas pessoas possam contribuir simultaneamente para o projeto. E através do GitHub, onde se pode fazer upload de seu projeto, para que outras pessoas possam vê-lo e contribuir para seu aprimoramento.
