Flask

Frameworks: Flask é microframework Python que auxilia na criação de aplicações web. Muitas vezes frameworks são confundidas com bibliotecas pois escolhem quando chamar seu codigo, porém bibliotecas também escolhem onde chamar seu código. Sendo uma framework mais simples que Django por exemplo ela usa de sua robustes e simplicidade para desenvolver aplicativos de forma simples, contudo não se deve pensar que uma framework insuficiente, podendo ser ultilizada para processos extremamente complexos.
